# Premium Zanzibar Tours & Safaris

Welcome to Premium Zanzibar Tours & Safaris, a modern travel website showcasing unforgettable Zanzibar tours, beach getaways, safari adventures, and premium hospitality experiences.

## Overview

This project is a responsive React + Vite website designed to promote travel packages, highlight destination highlights, and make it easy for visitors to explore and book amazing experiences in Zanzibar.

## Features

- Beautiful landing page with hero section and booking widget
- Featured tours and safari packages
- Destination, hotel, and beach highlights
- Testimonials and newsletter section
- Responsive design for desktop and mobile
- Smooth animations and modern UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM

## Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

```bash
npm install
```

### Run locally

```bash
npm run dev
```

The app will open in your browser at the local Vite URL.

### Build for production

```bash
npm run build
```
